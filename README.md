# 📰 React News App

A responsive React.js application that fetches real-time news using NewsAPI and displays categorized news in a clean and user-friendly interface.

---

## 🚀 Features

- Real-time news fetching using NewsAPI
- Category-based news (Technology, Sports, Business, etc.)
- Responsive design for mobile and desktop
- Clean and modern UI
- Fast and dynamic content loading

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML & CSS
- NewsAPI

---

## 🔑 Environment Variables

Create a `.env` file in the root directory and add:
REACT_APP_API_KEY_FUNCTION_COMP=your_api_key_here



## 📦 Installation & Setup

1. Clone the repository

git clone https://github.com/himanshu536-dev/NewsApp.git


2. Install dependencies

npm install


3. Start the development server

npm start
